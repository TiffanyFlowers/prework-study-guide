HTML

  • HTML stands for Hypertext Markup Language.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Heading tages are important to make pages accessible to users using assistive technologies.
  • HTML is made up of elements which work together to provide content and the basic framework or structure for all pages on the web.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS gives the content style.
  • CSS is like the style added on a house over the frame or foundation structure.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Three ways to style a webpage using CSS; inline CSS, internal CSS, and external CSS

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
    • Git stands for Global Information Tracker.
  • Git commands are entered in GitBash to write code to create applications.

JavaScript

  • JavaScript is a lightweight programming language.
  • JavaScript handles logic and allows users to interact with a website.
